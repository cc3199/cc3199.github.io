How to Batch Convert PDF to JPG

  1. Go to Windos>Actions and select the new action icon in the little actions window. Name your action something you will remember for said action, such as Pdf to Jpg.

  2. Make sure the record button is red, meaning it is recording. Then open a file in Photoshop that you wish to convert. Convert the file manually as you normally would (File>Save As and named properly with file extension as PDF and saving to the folder you wish to save it to). Then close your document (don’t close photoshop). Also, don’t stop recording until after you have closed your document.

  3. In Photoshop, go to Window>Actions and stop recording by selecting the square icon in the little actions window.

  4. Now you’re going to run the script you just recorded that will be applied to the rest of the files you wish to convert. Go to File>Scripts>Image Processor.

  5. The image processor has four steps:

    1. Select the images to process: select the folder that you want Photoshop to batch file convert.
    2. Select location to save processed images: select the folder that you want Photoshop to save the converted files.
    3. File type: select save as JPEG as it is selected by default. Then check the box ‘Resize to Fit’. You can then fill out the boxes that allow you to set the pixel dimensions that you want your converted files to be.
    4. Preferences: select ‘Run Action’ and then select the action that you recorded in the previous steps. Deselect the box ‘Include ICC Profile’.

  6. Select ‘Run’ and let Photoshop do all of the work for you!
