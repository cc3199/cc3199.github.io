How to Get Set Up

For general info about this, see the CollectionBulder docs.
  1. Create a repository. Please use CB docs for this step.

  2. Create a branch. For this step go to the MBMG Github Master code Then select the dropdown button that says "master".

  3. In the space in the dropdown, type the name of your collection and select to "create branch."
That's it! You now have an MBMG template for your collection. The next part will walk through how to customize the online collection to fit your collection.

Modifications to Config.yml File

For general info about this file, see the CollectionBulder docs
  1. To begin, locate and open the _config.yml file in the base of your repository. No matter which version you’re using, this file is used to configure the basics of your site.

  2. Mark off the area you do not want removed. This can be done by holding the left mouse button and dragging across the scan. The area to be removed will be “blacked out”.

  3. After marking off the area to be cropped, click on the check icon on the top center of the toolbar to apply changes. The cancel icon will cancel all changes made.
