Links to Documentation Description
Basic CollectionBuilder Docs This link leads to the CollectionBuilder website. Here, you can find out more about the different templates and access a step-by-step guide to getting started using CollectionBuilder starting with program installation until site deployment. Just select the 'Get Started' button on the navigation bar to begin the tutorial.
Setting up for CollectionBuilder There is a number of programs necessary to install for the CollectionBuilder to function properly.There are also three different versions of CollectionBuilder. MBMG collections use the Stand-Alone (SA)version which requires the most software dependencies. Here's how to install everything.
MBMG Modifications The basic CollectionBuilder docs will help get started and familiar with CollectionBuilder, but these docs here will walk through the specific changes to files necessary for the MBMG collection standard to keep consistency and functionality.
Adding and Deleting Web Pages Anyone familiar with website development should already know how this works. This tutorial will inform how to add or delete pages to the website from within the perspective of CollectionBuilder. It covers how to add or delete a page from the navigation bar, and how to add a page using a page link.
Metadata Modifications Shortcuts Most of MBMG collection metadata has some text capitalization issues when put into a website, such as the titles being in all-caps. This walks through shortcuts to change this to proper capitalization display using both Visual Studio Code and Microsoft Excel. A couple capitalization issues also has to do with the code itself that can be easily tweaked.
Favicon and Logos A favicon is the small icon that appears on the tab in the web browser. This tutorial covers how to change this icon as well as how to change the logos on CollectionBuilder to become the MBMG logos that look best and where they look best.
Deploying Collection Site CollectionBuilder's tutorials instruct how to deploy the finished website. The procedure for deploying the website for MBMG is identical, but this tutorial teaches how to move and where to move the deployed files. The deployed files can still be edited.
