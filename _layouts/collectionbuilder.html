Links to Documentation Description
CollectionBuilder Docs For reference, this link leads to the CollectionBuilder website. Here, you can find out more about the different templates and access a step-by-step guide to getting started using CollectionBuilder starting with program installation until site deployment. Just select the 'Get Started' button on the navigation bar to begin the tutorial.
Setting up for CollectionBuilder Begin here. There is a number of programs necessary to install for the CollectionBuilder to function properly.There are also three different versions of CollectionBuilder. MBMG collections use the Stand-Alone (SA) version which requires the most software dependencies. Here's how to install everything.
Building a Collection The basic CollectionBuilder docs will help get started and familiar with CollectionBuilder, but these docs here will walk through the specific changes to files necessary for the MBMG collection standard to keep consistency and functionality.
Advanced Customization This link leads to the CollectionBuilder documentation that covers how to modify the tabs on the navigation bar, what data appears on each object page, what data appears beneath the objects on the browse, map, and data page, as well as modifying the theme colors. Be sure to follow any special instructions for SA users and ignore any GH special instructions.
Customizing Home and About Pages You can also customize and change how the home page and about page looks. The about page can explain in bigger detail what your collection is all about or keys to understanding maps, or other items in the collection, or anything else that would be pertinent to the collection. The home page may need altering if certain data is irrelevant, such as a timeline, or titles that need to be changed.
Metadata Modifications Shortcuts A lot of MBMG collection metadata has some text capitalization issues when put into a website, such as the titles being in all-caps. This walks through shortcuts to change this to proper capitalization display using both Visual Studio Code and Microsoft Excel. A couple capitalization issues also has to do with the code itself that can be easily tweaked.
Deploying Collection Site CollectionBuilder's tutorials instruct how to deploy the finished website. The procedure for deploying the website for MBMG is identical, but this tutorial teaches how to move and where to move the deployed files. The deployed files can still be edited.
