Photoshop Techniques for Cleaning Files

You may use any, all, or none of these techniques for cleaning files based on the condition of the scanned file. The buttons in red are steps that should be completed when cleaning EVERY file. Please see a staff member if you have any questions.

Duplicate Layers

  1. Photoshop works in 'layers', a way to organize different steps and edits. The layers panel is located in the lower-right corner of the window.
  2. When an image is first opened, you will notice that the image is 'locked' in the right side of the layer, indicated with a lock symbol. This is the original copy.


    3. You should ALWAYS duplicate the original layer. Use the original layer as a reference during the optimization process or to start over with the page.
  3. Right-click on the locked image and choose Duplicate Layer. This will make a copy of the original.


  4. Turn off the 'eye' on the background layer by clicking it so that only the layer saying 'background copy' has an eye icon next to it. This will be the copy you will be working with (seeing).


Rotate Images

Rotating an image in this step will rotate visible layers
    To rotate the image, go to the menu on the top left side of the screen. Navigate to Image>Image Rotation and there is the option to rotate an image 180 or 90 degrees either direction; select the option needed.
    • There is also an 'Arbitrary' option, which allows you to manually choose the degree to which to rotate if 180 or 90 is not what is needed. Click here to learn how to straighten an image.


Crop an Image

Before making any edits to your document, be sure to duplicate the image layer.
  1. The crop tool is located in the toolbar on the upper left side of the screen and is used to trim any area off of an image, such as extra white-space.

  2. Mark off the area you do not want removed. This can be done by holding the left mouse button and dragging across the scan. The area to be removed will be “blacked out”.

  3. After marking off the area to be cropped, click on the check icon on the top center of the toolbar to apply changes. The cancel icon will cancel all changes made.

Straighten an Image

Before making any edits to your document, be sure to duplicate the image layer.
  1. Select the 'Ruler' tool on the left-side toolbar. It can be found by right-clicking on the 'Eye-Dropper Tool'.


  2. Click and drag along a crooked line on the image and then select 'Straighten Layer' at the top toolbar. If your layer doesn't have any lines, but has text instead, drag a line underneath a line of text as an imaginary line.


  3. This will often change the canvas size as you will see the checkered background around the edges. Check your canvas size, especially for standard dimension documents such as 11x17, 8.5x11, and so on.


  4. Select the check mark in the tool bar to confirm the action.

Dodge and Burn Tools

Before making any edits to your document, be sure to duplicate the image layer.
The dodge tool is a brush used to lighten where brushed, the burn tool is a brush used to darken where brushed. The burn tool is commonly used to darken text to make it more legible.
  1. The Dodge and Burn tool is located on the left-side toolbar. Right-click on the icon to reveal more tool options.Be sure to either have the burn or dodge tool selected.


  2. You can adjust the exposure level by typing in the amount of exposure you want or by using a slider. The slider can by accessed by clicking the arrow on the right side of the exposure input area.

  3. You can select which range you want (shadows, midtones, or hightlights) to change. You will have to experiment with different ranges to fully understand you they work for whichever effect that you desire.

Clone Stamp Tool

Before making any edits to your document, be sure to duplicate the image layer.
The Clone Stamp tool is a brush that allows you to copy an area and clone that copied area to another location on the canvas. This tool is extremely helpful when trying to replace lines and letters.
See other similar tools (such as the selection, spot healing brush, and the patch tool) to pick the most efficient for the situation. Sometimes a combination of these tools is necessary to get the desired outcome.
  1. The Clone Stamp tool is located on the left-side toolbar. Right-click on the icon to reveal more tool options. Be sure to have the Clone Stamp Tool selected.

  2. To Clone Stamp an unwanted area, hover over the area you want to clone to another area. Hold down the ALT key and left-click. This will select the clone area.

  3. Next, move over the area to be covered up and hold the left button down while brushing over the desired area.

Selection Tool and Fill

Before making any edits to your document, be sure to duplicate the image layer.

Marquee Tool

The Marquee tool is a selection tool with some handy filling options, such as content-aware, foreground, and background. Content-aware filling considers the content outside the selection to the fill with that content. Foreground filling uses the top color in the color squares in the left-side toolbar. The background fill uses the bottom color in the color squares in the left-side toolbar.
See other similar tools (such as the clone stamp, spot healing brush, and the patch tool) to pick the most efficient for the situation. Sometimes a combination of these tools is necessary to get the desired outcome.
  1. Select the Rectangular Marquee Tool, which is located on the left-side toolbar.

  2. Select the area to be corrected by clicking and dragging a square across the area to be repaired.

  3. With your cursor inside the selected area, right-click to access the sub-menu and select 'fill'.

  4. A small window will pop up. In the first drop-down, select 'Content-Aware'.

Color Range Selection

Color Range selection uses the "eyedropper" to select same colored pixels throughout the document.
  1. First, you will need to select the color of the document and save it as the foreground color. To do this, select the foreground color in the squares in the left-side toolbar. When the window pops up, use your cursor (which will now take the shape of an eyedropper) to select anywhere on the background of the document. This will be the color used to fill in parts of the document that are not wanted to be replaced with the document color to blend in.


  2. Locate Color Range by navigating to 'Select>Color Range'.


  3. When the window pops up, use your cursor (which will now take the shape of an eyedropper) to select the section you want to select. Do not press 'OK' yet.


  4. In the window, select the eyedropper with the little plus symbol next to it. Then click and drag over a majority of the area you want to select. This is just to ensure that all of the colored pixels in this range are selected. You may unclick and click and hold again in a different area, or even click multiple times to ensure every color you want replaced is selected.Then select 'OK'.


  5. You should see lines around the area of the colors you selected. Now select the 'Marquee' tool then right-click with your cursor in the selected area and select 'Fill'.


  6. Once the selection is filled, you may or may not see some spots that were missed. You can repeat the process over agin until you get the result you are looking for.

Spot Healing Brush Tool

Before making any edits to your document, be sure to duplicate the image layer.

The Spot Healing Brush Tool uses content-aware to fill in the spots brushed over with this tool. It is useful for quickly covering small areas with imperfections or cleaning up edges of documents.

See other similar tools (such as the clone stamp, marquee, and the patch tool) to pick the most efficient for the situation. Sometimes a combination of these tools is necessary to get the desired outcome.

  1. Select the Spot Healing Brush Tool, which is located on the left-side toolbar. You can right-click this tool to reveal more similar tool options.

  2. Brush over the area with the imperfection to 'heal' it.

  3. Continue brush use for any other imperfections. Notice the difference at the tops of the images below for a before and after.

Patch Tool

Before making any edits to your document, be sure to duplicate the image layer.

The patch tool is quite similar to the healing spot tool, except that the area to be fixed is selected and then dragged to an area that would 'patch' the selection area. The patch tool tries to retain the tone and color of the location to be patched

See other similar tools (such as the clone stamp, marquee, and the spot healing brush tool) to pick the most efficient for the situation. Sometimes a combination of these tools is necessary to get the desired outcome.

  1. Select the Patch Tool, which is located on the left-side toolbar usually accessed by right-clicking on the Spot Healing Brush Tool and selecting Patch Tool.

  2. Draw a selection circle around the area you want to fix.

  3. Left-click in the selected circle and drag to an area that with good texture (Color doesn't necessarily matter too much because the patch tool will attempt to keep as much of the local color as possible while replacing the texture.). Then release the click.

  4. Hold CTRL-D to deselect the area.

  5. You may notice that the patch tool doesn't work too well with colors. This is what happens when I try to fix the edge of the page: The edge was blurred rather than filled in. This is a job for either the spot healing brush tool or the clone stamp.


Check Image Size

  1. As a quick check of your canvas size, look to the very lower-left of the Photoshop window. This small box displays the image/canvas size and the ppi (pixels per inch). Check that these numbers match the original document size you started with before applying any edits. Standard document sizes are 8.5x11 (letter), 8.5x14 (legal), and 11x17 (ledger). Many maps most likely will not adhere to these standards, so check what the original scanned dimensions were. In this case, the file's canvas size has changed over the course of editing.

  2. To fix this, navigate to 'Image' on the top navigation bar, then to 'Canvas Size'.

  3. In the next window, change to the appropriate dimensions.

You should ALWAYS check the canvas size before saving and closing your file. This should also be the very last thing that you do before saving and closing the file.
