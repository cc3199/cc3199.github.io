How to Duplicate Layers

  1. Photoshop works in 'layers', a way to organize different steps and edits. The layers panel is located in the lower-right corner of the window. Duplicating the original layer should ALWAYS be the first thing you do. This ensures that if any mistakes are made, the original layer is preserved.
  2. When an image is first opened, you will notice that the image is 'locked' in the right side of the layer, indicated with a lock symbol. This is the original copy.


  3. Right-click on the locked image and choose Duplicate Layer. This will make a copy of the original.


  4. Turn off the 'eye' on the background layer by clicking it so that only the layer saying 'background copy' has an eye icon next to it. This will be the copy you will be working with (seeing).


How to Rotate Images

    To rotate the image, go to the menu on the top left side of the screen. Navigate to Image>Image Rotation and there is the option to rotate an image 180 or 90 degrees either direction; select the option needed.
    • There is also an 'Arbitrary' option, which allows you to manually choose the degree to which to rotate if 180 or 90 is not what is needed. Click here to learn how to straighten an image.


How to Crop an Image

Before making any edits to your document, be sure to duplicate the image layer.
  1. The crop tool is located in the toolbar on the upper left side of the screen and is used to trim any area off of an image, such as extra white-space.

  2. Mark off the area you do not want removed. This can be done by holding the left mouse button and dragging across the scan. The area to be removed will be “blacked out”.

  3. After marking off the area to be cropped, click on the check icon on the top center of the toolbar to apply changes. The cancel icon will cancel all changes made.

How to Straighten an Image

Before making any edits to your document, be sure to duplicate the image layer.
  1. After selecting the crop tool, a small leveler icon will appear on the top toolbar with 'Straighten' next to it.


  2. Click and drag along a crooked line on the image and it will auto straighten the image using the line drawn. In this case, this will rotate the image 1.8 degrees clockwise.


  3. Select the check mark in the tool bar to confirm the action.

The Dodge and Burn Tools

Before making any edits to your document, be sure to duplicate the image layer.
The dodge tool is a brush used to lighten where brushed, the burn tool is a brush used to darken where brushed. The burn tool is commonly used to darken text to make it more legible.
  1. The Dodge and Burn tool is located on the left-side toolbar. Right-click on the icon to reveal more tool options.Be sure to either have the burn or dodge tool selected.


  2. You can adjust the exposure level by typing in the amount of exposure you want or by using a slider. The slider can by accessed by clicking the arrow on the right side of the exposure input area.

  3. You can select which range you want (shadows, midtones, or hightlights) to change. You will have to experiment with different ranges to fully understand you they work for whichever effect that you desire.

The Clone Stamp Tool

Before making any edits to your document, be sure to duplicate the image layer.
The Clone Stamp tool is a brush that allows you to copy an area and clone that copied area to another location on the canvas. This tool is extremely helpful when trying to replace lines and letters.
See other similar tools (such as the marquee, spot healing brush, and the patch tool) to pick the most efficient for the situation. Sometimes a combination of these tools is necessary to get the desired outcome.
  1. The Clone Stamp tool is located on the left-side toolbar. Right-click on the icon to reveal more tool options. Be sure to have the Clone Stamp Tool selected.

  2. To Clone Stamp an unwanted area, hover over the area you want to clone to another area. Hold down the ALT key and left-click. This will select the clone area.

The Marquee Tool and Content-Aware Fill

Before making any edits to your document, be sure to duplicate the image layer.
The Marquee tool is a selection tool with some handy filling options, such as content-aware. Content-aware filling considers the content outside the selection to the fill with that content.
See other similar tools (such as the clone stamp, spot healing brush, and the patch tool) to pick the most efficient for the situation. Sometimes a combination of these tools is necessary to get the desired outcome.
  1. Select the Rectangular Marquee Tool, which is located on the left-side toolbar.

  2. Select the area to be corrected by clicking and dragging a square across the area to be repaired.

  3. With your cursor inside the selected area, right-click to access the sub-menu and select 'fill'.

  4. A small window will pop up. In the first drop-down, select 'Content-Aware'.
