How to Batch Convert PDF to JPG

  1. Go to File>Scripts>Image Processor.

  2. The image processor has four steps:

    1. Select the images to process: select the folder that you want Photoshop to batch file convert.
    2. Select the location to save processed images: select the folder that you want Photoshop to save the converted files.
    3. File type: select save as JPEG as it is selected by default. Then check the box ‘Resize to Fit’. You can then fill out the boxes that allow you to set the pixel (not inches) dimensions that you want your converted files to be. (8.5in x 11in is equal to 2550px x 3300px)
    4. Preferences: deselect everything

  3. Select ‘Run’ and let Photoshop do all of the work for you!
