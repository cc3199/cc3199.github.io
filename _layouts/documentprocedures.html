How to Batch Convert Single-Page PDF to JPG

If you haven't already installed Document Cleaner 2.0, please do so before proceeding beyond this point.
  1. Open the files you want to convert in Photoshop. This works best if you're converting a pdf file that contains multiple pages.

  2. Go to File>Automate>Batch.

  3. Fill out the form as indicated in the image below:

    1. Under file naming, use the file naming conventions to name your files.

  4. Select ‘Run’ and let Photoshop do all of the work for you!

How to Conduct Batch Edits with Macros

If you haven't already installed Document Cleaner 2.0, please do so before proceeding beyond this point.
  1. Go to Windows>Actions.

  2. Select the new action icon in the little actions window

  3. Name your action something simple that describes your action and then press "Record."

    • Open a file in Photoshop that you wish to convert. Convert the file manually as you normally would. Keep in mind that it has to be done perfectly when recording; no undoing is allowed.
    • Then close your document (don’t close photoshop). Also, don’t stop recording until after you have closed your document.

  4. In Photoshop, go to Window>Actions and stop recording by selecting the square icon in the little actions window.

  5. Finally, select this macro from the list and press "Play."
